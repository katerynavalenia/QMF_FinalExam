# QMF Final Exam: Counterfactual Inflation Analysis for Ukraine

**What If Ukraine Had Been Part of the Euro Area?**

Quantitative Methods in Finance — Final Exam 2025–2026
Master 2 Research, Université Paris 1 Panthéon-Sorbonne

## Overview

This project constructs a counterfactual inflation path for Ukraine under the hypothetical scenario of Euro Area membership, using structural VAR and factor model methods.

### Methods
- **Primary**: Bayoumi–Eichengreen bivariate SVAR with Blanchard-Quah long-run identification + demand-shock replacement
- **Robustness**: Ciccarelli–Mojon common factor model (PCA on EA-11 inflation panel)

### Technical Improvements
- Stationarity handling: ADF tests with first-differencing and IRF back-transformation
- Lag order: VAR(12) minimum with seasonal dummies for monthly data
- Bootstrap confidence intervals (n=500) for IRFs and counterfactual paths
- Shock replacement: UA demand shocks replaced with EA-scaled equivalents (Bayoumi–Eichengreen approach)

### Key Findings
- Ukraine's monetary sovereignty was **time-varying**: dollar pegs (2000–2008, 2009–2014), forced floats (2008–09, 2014–15), inflation targeting (2016–2021), and wartime pegs (2022+)
- Euro Area membership would have **reduced the 2014–15 inflation spike** (60.9% → 26.6%) by removing the exchange-rate devaluation channel
- During peacetime pegs, the treatment effect is near zero (Ukraine already lacked monetary independence)
- The 2022 war was supply-shock driven, so EA membership would have provided limited benefit

## How to Run

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the full pipeline:**
   ```bash
   python main.py
   ```
   This executes all 5 phases: data loading, Part A analysis, SVAR estimation, factor model, and figure generation. All outputs are written to `output/`.

3. **View results in the notebook:**
   The file `qmf_counterfactual_ukraine.ipynb` contains the complete narrative with all figures, tables, and interpretation. Open it directly in Jupyter.

4. **Run individual modules:**
   ```bash
   python src/01_data.py        # Data loading only
   python src/02_part_a.py      # Part A only
   python src/03_svar.py        # SVAR only
   python src/04_factor.py      # Factor model only
   python src/05_figures.py     # Figures only
   ```
